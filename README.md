# Vantage - Simple FPS camera extension

This project rotates a camera compoent based on inputs in a FPS style fashion.

To use this extension:
* add https://github.com/Jhonnyg/vantage/archive/refs/heads/master.zip as a project dependency
* add the 'vantage_component' script to a camera
* set the supported input bindings in your game.input_binding or copy and set the provided input_binding file from this repository
